Information gathering
Information Gathering is the first and foundation step in the success of penetration testing. The more useful information you have about a target, more you able to find vulnerabilities in the target and hence able to find more serious problems in the target by exploiting them (to demonstrate).
Find subdomains
Discover subdomains and determine the attack surface of an organization
Website Recon
Find which web technologies your website is using
Google Hacking
Use advanced search operators to find juicy information about your websites
Find Virtual Hosts
Discover the virtual hosts configured on a given IP address
Web Application Testing
Web Penetration Testing is as the name suggestions, a penetration test that focuses solely on a web application rather than a network or company.
SQLi
Discover SQL Injection vulnerabilities in web applications
XSS
Test if your web application is vulnerable to Cross-Site Scripting
URL Fuzzer
Discover hidden files and directories
Web Server Scan
Discover common web application vulnerabilities and server configuration issues
Infrastructure Testing
An infrastructure penetration test is a proven method of evaluating the security of your computing networks, infrastructure and application weaknesses by simulating a malicious attack.
TCP Port Scan
Discover open TCP ports, detect service version and operating system
UDP Port Scan
Discover open UDP ports, detect service version and operating system
Whois Lookup
Find information about an Internet resource (domain name, IP address)
